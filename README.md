# Internship Offer Portal

## Overview
This project is a responsive Internship Offer Portal built using HTML, CSS, and JavaScript. It allows users to view internship details and interact with the portal through a clean and user-friendly interface.

## Features
- Responsive webpage design
- Internship details display
- Interactive user interface
- Form validation using JavaScript
- Add and delete task functionality

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
- `index.html` – main webpage structure
- `style.css` – styling and layout
- `script.js` – interactive functionality

## Purpose
This project was created as part of my internship Task 2 to improve my frontend development skills by building a practical web application.

## How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
